## Features

- **Dark Twitch-themed design** — Purple-accented dark UI matching the streaming ecosystem
- **Animated sections** — Smooth scroll-triggered animations with Framer Motion
- **Internationalization** — Full English & Turkish language support (i18n)
- **Responsive** — Mobile-first design that works on all devices
- **Fast** — Built on Next.js 16 with optimized fonts and images

## Tech Stack

```
Next.js 16  ─────>  App Router, Server Components
Tailwind v4 ─────>  Utility-first styling
Framer Motion ───>  Scroll & entrance animations
TypeScript  ─────>  End-to-end type safety
i18n  ───────────>  EN / TR translations
```

## Getting Started

```bash
git clone https://github.com/kaankuzu1/growthlanding.git
cd growthlanding
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
